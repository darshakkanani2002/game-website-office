Installation

After doing basic customization , now you are ready to publish your game on server( Hosting ) / Google play store / iOS App Store

  1. - Host game on server ( hosting ) :- This is very simple , you need your own web hosting with domain- Put all files "index.html, css folder, assets folder" files in your server public_html or root folder or in any sub folder, thats it, now just simply access your domain in browser and your game is ready to play ;)

    2- Publish game on google play store :- This is very simple, if you have basic knowledge of android studio then start android studio and make a webview screen and connect with your website domain in which you uploaded game files, now setup icon, package name, splash screen, admob and then build apk. Thats it, your android game is ready now you can submit your app in google play store.
